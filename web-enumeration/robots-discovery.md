# Web Enumeration

After accessing the web application, the next step was to look for hidden files and directories.

One of the common files checked during web enumeration is **robots.txt**.

URL:
http://<TARGET-IP>/robots.txt

Inside this file several interesting entries were discovered.

These entries revealed hidden resources that were not directly visible on the website.

While exploring the discovered paths, the first flag was located.

## Key 1 Discovery

The first key was discovered during the enumeration of the resources listed inside the robots.txt file.

This demonstrates how important it is to always check common files during web enumeration, as they may reveal sensitive information or hidden paths.
