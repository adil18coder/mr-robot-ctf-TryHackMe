# Mr Robot CTF – TryHackMe Writeup

## Overview

This repository contains my walkthrough and notes for the **Mr Robot CTF** room on TryHackMe.

The challenge is inspired by the Mr. Robot TV series and focuses on several fundamental penetration testing skills including:

- Enumeration
- Web application discovery
- Credential harvesting
- Privilege escalation

The objective of the room is to capture three flags while compromising the target system.

---

## Skills Practiced

- Network scanning
- Web enumeration
- WordPress discovery
- Password cracking
- Linux privilege escalation

---

## Tools Used

- Nmap – network scanning
- Gobuster – directory enumeration
- Hydra – password brute forcing
- Netcat – reverse shell handling
- Linux enumeration commands
